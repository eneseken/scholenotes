Cisco Packet Tracer ile IoT bağlantıları

Eğer IoT bileşenlerini kurarken hata alıyorsan bu yazımı mutlaka okumalısın

İlk Öncelikle kullanacağımız IoT bileşenlerini tanıyalım ;

Cihazları tanıdığına göre bu adımları sırasıyla takip et

  • 1- IoT Cihazlarını Koy.
  • 2- IoT Cihazlarını İsimlendir.
  • 3- IoT Cihazlarının Config > Wireless0 > SSID kısmına HomeGateway gir.
  • 4- (IoT Cihazları) Config > Settings > IoT Server > HomeGateway
  • 5- IoT Cihazlarını Advanced > I/O Config > Network Adapter > PT-IOT-NM-1W olarak ayarla.
  • 6- Eğer IoT Cihazlarının Advanced > I/O Config > Programing kısmında Starting Appliance (JavaScript)... yazmıyorsa buraya tıkla
  • 7- HomeGateway cihazını koy. (Iot Cihazları arasındaki bağlantıyı gör)
  • 8- Uzaktan erişim sağlamak için bir cihaz koy. (Laptop/Telefon)
  • 9- Cihazın Config > Wireless0 > SSID kısmına HomeGateway yaz.
  • 10- Cihazın Desktop > IoT Monitor kısmından Iot Cihazlarını kontrol et.
